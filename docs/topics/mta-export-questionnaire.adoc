= Exporting a questionnaire

.Procedure

Export a questionnaire:

. In the *Administration* view, select *Assessment questionnaires*.
. Select the questionnaire.
. Click the Options menu {kebab}.
. Select *Export*.
. Select the location of the download.
. Click *Save*.
