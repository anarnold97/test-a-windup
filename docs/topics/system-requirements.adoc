= System requirements

* One of the following Java Development Kits (JDKs):

** OpenJDK 1.8
** OpenJDK 11
** Oracle JDK 1.8
** Oracle JDK 11

* 8 GB RAM
* If you are installing on macOS, the value of `maxproc` must be `2048` or greater.
